# AI Fashion Marketing Content Generator

An AI-powered fashion marketing web application that generates high-quality promotional visuals using Retrieval Augmented Generation (RAG) and FLUX image generation.

## Features

- Brand-aware marketing image generation
- FAISS-based semantic search (RAG pipeline)
- AI image generation via Replicate (FLUX model)
- Clean Streamlit web interface
- Image history gallery

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set `REPLICATE_API_TOKEN` environment variable
4. Run: `streamlit run app.py`

## Deployment

This app is configured for Hugging Face Spaces deployment.
